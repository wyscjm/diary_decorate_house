## 水循环
1. 阳台要有水
2. 卫生间
3. 餐厅
其他地方不需要。

## 电循环
1. 三个卧室门口及床位处要装一个双控开关。
2. 餐厅桌子底下要有一个插头，接电气的。
3. 门口有个开关控制餐厅和客厅的灯
4. 走道有个灯，且是双控的。两端都可以开关，
   或者智能化，可以接一个智能灯
5. 空调走线单独走
